Kubernetes Cluster Management

Project Overview

This project involved building a full-fledged CI/CD pipeline from scratch, ensuring that the entire process, from code commit to deployment, is automated and scalable using Jenkins and AWS CodePipeline.

Technologies Used

AWS CodePipeline Jenkins Docker Terraform

Project Goals & Achievements

This project aimed to automate the entire deployment lifecycle, from code commit to production. Key achievements include:

  • Automating deployments using Jenkins and AWS CodePipeline
  • Integrating Docker containers for consistent deployment across environments
  • Using Terraform for infrastructure as code to provision AWS resources

Challenges & Solutions

During the project, I faced several challenges, including setting up reliable webhook triggers for Jenkins and managing multiple deployment stages. Solutions involved customizing the Jenkins pipeline scripts and using AWS Lambda for automatic trigger management.
